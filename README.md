## Commandes
- new_set(armor_name,weapon_name):
    Enregistre un nouveau set au nom de l'utilisateurs.

    Parametres:
    - armor_name
        Nom du set de l'armure

        Type : str
    - weapon_name
        Nom de l'arme

        Type : str

- get_set(number):
    Renvoi le set choisi par l'utilisateur.

    Parametres:
    - number
        Numéro du set

        type : int

- get_sets():
    Renvoi tout les sets enregistrés au nom de l'utilisateur.
